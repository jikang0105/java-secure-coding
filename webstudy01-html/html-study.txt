	HTML : HyperText Markup Language
		  웹 문서의 화면 구성을 정의하는 언어
		 참고사이트 : www.w3schools.com
	Javascript : 웹 문서의 동작을 담당한다.(최근에는 서버 사이드 프로그래밍도 지원한다)
	CSS : 웹 문서의 디자인 스타일을 담당한다.
	XML : Extensible Markup Language -> 주로 웹어플리케이션 설정 정보를 명시할 때 사용한다
										데이터 통신용으로도 사용되었지만 최근에는 JSON을 이용한다.
		  
		 <!DOCTYPE html> <!-- html 주석 : html 5 버전 문서임을 명시하는 선언부 -->

		 </head>
		 <body> <!-- 본문 화면의 디자인 구성: 웹 화면을 표현 -->
