**요구사항**
: 사용자는 계좌 개설이 가능하다 
계좌 개설시 계좌번호, 계좌주명, 비밀번호, 잔액정보가 저장되어야 한다 
최초 계좌 개설시에는 초기 납입액 1000원 이상이 되어야 한다 
계좌번호는 유일해야 하고 시스템에서 자동발급되도록 관리한다 
잔액확인은 계좌번호가 존재해야 하고 비밀번호가 일치해야 한다 
입금액, 출금액, 계좌이체액은 모두 0원을 초과해야 한다
입금시에는 계촤번호,비밀번호 일치해야 한다 
출금시에는 계좌번호, 비밀번호, 잔액확인 절차가 필요하다 
계좌이체시에는 송금자 및 수금자의 계좌가 존재해야 하고 
이체액은 0원을 초과해야 하며 송금자의 비밀번호가 일치해야 한다 
또한 송금자의 잔액 확인이 필요하다 

- 요구분석 
	계좌개설
	잔액조회
	입금
	출금
	계좌이체

- 설계 
   데이터베이스 -> SQL DDL : CREATE , 제약조건 , 데이터타입 
   어플리케이션 -> UML ( Class Diagram ) 
   					 필요한 클래스 추출 -> 주요메서드 설계 
   					 
   분석, 설계 소요시간 
   구현 소요시간 					 
   
   풀이 순서 
   계좌 개설 -> 입금 -> 출금 -> 계좌이체
